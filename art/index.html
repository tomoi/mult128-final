For this project I wanted to start with the scattering. Since I am using a laptop to make this, I wanted to keep the strees on my machine down to a minimum while in the viewport. This inspired me to use geometry nodes to dynamically change the amount of scattered objects based on the distance to the camera object.
I also have it set so it will switch between different objects depending on if it is in the viewport or rendering, so I can keep the number of faces down.
